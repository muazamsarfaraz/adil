Accessibility Statement

Our commitment to making AskAdil accessible to everyone.

Our Commitment

AskAdil is committed to ensuring digital accessibility for people with disabilities. We continually improve the user experience for everyone and apply the relevant accessibility standards to achieve this.

We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.2 at Level AA. These guidelines explain how to make web content more accessible to people with a wide range of disabilities, including visual, auditory, physical, speech, cognitive, language, learning, and neurological disabilities.

Measures Taken

We have taken the following measures to ensure accessibility on the AskAdil landing page:

  • Sufficient colour contrast ratios (minimum 4.5:1 for normal text, 3:1 for large text) across all text and background combinations
  • Descriptive alternative text on all meaningful images
  • Visible focus indicators on all interactive elements for keyboard navigation
  • ARIA landmark roles and labels on navigation, main content, and footer regions
  • A skip-to-content link as the first focusable element on each page
  • Logical tab order for keyboard-only navigation
  • Semantic HTML elements (header, nav, main, section, footer)
  • Language attributes on foreign-language text (Arabic)
  • Respect for user motion preferences via prefers-reduced-motion
  • Responsive design that works across screen sizes and zoom levels up to 200%

Known Limitations

Despite our best efforts, some areas may have limited accessibility:

  • AI-generated placeholder images: The current hero and story card images are AI-generated placeholders. While they have descriptive alt text, they will be replaced with genuine editorial photography that better represents the British Muslim community.
  • Third-party content: External links to legislation.gov.uk, Tell MAMA, ACAS, and other third-party resources are governed by those organisations' own accessibility policies.
  • Mobile navigation: On small screens, the navigation links are currently hidden. A fully accessible mobile menu is planned for a future update.

Feedback and Contact

We welcome your feedback on the accessibility of the AskAdil website. If you encounter any accessibility barriers or have suggestions for improvement, please contact us:

  • Email: accessibility@askadil.org
  • Organisation: Muslim Council of Britain

We aim to respond to accessibility feedback within 5 working days.

Compatibility

AskAdil is designed to be compatible with the following assistive technologies:

  • Screen readers (NVDA, JAWS, VoiceOver, TalkBack)
  • Screen magnification software
  • Speech recognition software
  • Keyboard-only navigation

Standards

This website aims to conform to WCAG 2.2 Level AA. WCAG 2.2 is the current international standard for web accessibility, published by the World Wide Web Consortium (W3C).
