## 4. Data Collection Form

For each firm contacted, collect and record in the JSON directory:

### Firm Verification

| Field | Question |
|-------|----------|
| `outreach_status` | contacted / responded / verified / declined |
| `consent_to_list` | true / false / pending |
| `outreach_date` | Date of first contact |
| `outreach_notes` | Any relevant notes from conversation |

### Service Capabilities

| Field | Question |
|-------|----------|
| `free_consultation` | Do you offer a free initial consultation? (yes/no/duration) |
| `no_win_no_fee` | Do you offer no-win-no-fee? (yes/no/which areas) |
| `legal_aid` | Do you hold a legal aid contract? (yes/no/which areas) |
| `languages_verified` | What languages does your team speak? |
| `preferred_referral_method` | How should AskAdil refer clients? (phone/email/form/online booking) |
| `referral_email` | Dedicated email for AskAdil referrals (if different from general) |
| `preferred_case_types` | Which case types do you want referred? |
| `excluded_case_types` | Any case types you don't want? |
| `capacity` | Can you take on new clients currently? (yes/limited/waitlist) |
| `geographic_limit` | Any geographic restrictions on clients? |
| `response_time` | How quickly do you typically respond to enquiries? |

## 7. Outreach Team & Responsibilities

| Role | Responsibility |
|------|---------------|
| **Outreach Lead** | Send emails, make calls, track responses, update JSON |
| **MCB Liaison** | Provide MCB letterhead/endorsement for credibility |
| **Technical Lead** | Update directory data as responses come in, embed in RAG |
| **Legal Review** | Review any commercial arrangements before agreeing |

### MCB Endorsement

An MCB-branded introductory letter significantly increases response rates. Request:
- MCB logo for email signature
- Brief endorsement paragraph from MCB Secretary General
- Permission to say "in association with the Muslim Council of Britain"
- MCB to share the directory call-to-action via their channels (newsletter, social media, affiliate mosques)

---

## 8. Risk Mitigation

| Risk | Mitigation |
|------|-----------|
| Low response rate | Phone follow-up after email. MCB endorsement increases credibility. LinkedIn InMail as alternative channel. |
| Firms decline to be listed | Respect decision. Ask if they'd reconsider if/when directory is established. Record reason for decline. |
| Inaccurate contact details | Cross-reference with SRA register before contacting. Call firm switchboard if email bounces. |
| Firm wants payment to be listed | Explain free model. If they insist, note for future premium tier discussion. |
| Negative reaction to concept | Listen to concerns. Emphasise community benefit. Offer to remove if they change mind later. |
| GDPR concerns from firms | All data used is publicly available on firm websites. Listing requires explicit consent. Right to removal honoured promptly. |
| Outreach fatigue | Batch waves over 6 weeks. Don't contact more than 10 firms per week. |
